## 3월에 주문이 존재하는 고객들 조회하기

```sql
SELECT  *
FROM    M_CUS T1
WHERE   EXISTS(
            SELECT  *
            FROM    T_ORD A
            WHERE   A.CUS_ID = T1.CUS_ID
            AND     A.ORD_DT >= TO_DATE('20170301','YYYYMMDD')
            AND     A.ORD_DT < TO_DATE('20170401','YYYYMMDD')
            );
```

반대로 3월에 주문이 한 건도 없는 고객을 조회해야 한다면 NOT EXITS를 사용하면 된다.
